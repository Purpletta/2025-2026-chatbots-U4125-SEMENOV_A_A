# Habit tracker — Telegram-бот (лабораторная работа 1)

Репозиторий курса [Vibe Coding: AI-боты для бизнеса](https://github.com/itmo-ict-faculty/vibe-coding-for-business).

Бот [@ItmoHabitTrackerBot](https://t.me/ItmoHabitTrackerBot) помогает вести привычки: добавление, список, отметка за сегодня, удаление, простая статистика. **Управление — кнопками** (нижняя клавиатура и inline-кнопки выбора привычки). Текстом можно ввести только **название новой привычки** после нажатия «Создать привычку». Данные хранятся в `data.json` (создаётся автоматически, в репозиторий не коммитится).

## Требования

- Python 3.10+
- Токен бота от [@BotFather](https://t.me/BotFather)

## Установка и запуск

```bash
cd 2025-2026-chatbots-U4125-SEMENOV_A_A
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# Откройте .env и укажите BOT_TOKEN=...
python bot.py
```

Бот работает в режиме **long polling** (сервер с интернетом нужен только машине, где запущен процесс).

## Интерфейс

| Действие | Как |
|----------|-----|
| Первый запуск | `/start` в чате с ботом — появится меню кнопок |
| Создать привычку | Кнопка «Создать привычку» → затем **одним сообщением** название (или «Отмена») |
| Список | «Список» |
| Отметить сегодня | «Отметить» → нажать inline-кнопку с названием привычки |
| Статистика | «Статистика» |
| Удалить | «Удалить» → выбрать привычку кнопкой |
| Справка | «Помощь» |

Произвольный текст вне режима создания привычки бот не использует (подскажет нажать кнопки). Дата «сегодня» — **Europe/Moscow**.

## Безопасность

- Не публикуйте токен и не коммитьте файл `.env`.
- Если токен утёк — в BotFather: `/revoke` для бота и выпустите новый токен.
